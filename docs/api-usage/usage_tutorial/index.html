API Tutorial

RDFShape API is a public API that can be used as a playground for RDF, Shape Expressions, SHACL, etc.

  • Example

Data + Schema#

  • If the Schema option is selected, then RDFShape API validates both the syntax and the shape of RDF graphs.
  • The Schema can be entered by URI, by file or by text input.
  • Once the validation has finished, the system can generate a shareable permalink with the result.
  • For visualizations, the system can also generate a link for embedding the visualizations.

Data + Schema + Node#

  • In this option, the system asks for a node in the RDF Graph that will act as the pointed node in the graph to start the validation process

Data Conversions#

  • This option allows to convert between different RDF formats.
  • The available formats are:
    • Turtle
    • RDF/XML
    • RDF/JSON
    • N-Triples

Schema Conversions#

  • This option can be used to convert different representations of schema data.
  • The available formats are:
    • ShExC (Shape expression compact syntax)
    • The RDF formats: Turtle, RDF/XML, RDF/JSON, N-TRIPLES, etc.

Further info#

RDFShape API is based on SHaclEX, a Shape Expressions processor which can also be used as a command line tool.
